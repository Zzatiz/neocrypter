A next-generation file crypter and packer toolkit. The full project is distributed as `neocrypter.rar` — extract to get started.

## Overview

neocrypter is a file encryption and obfuscation tool designed to pack and protect executables. It builds on the concepts from zzcrypter with an expanded feature set for file evasion and obfuscation.

## Contents

The `neocrypter.rar` archive contains the full build, source, and any bundled stubs.

## Usage

Extract `neocrypter.rar` and follow the instructions in the bundled documentation.
